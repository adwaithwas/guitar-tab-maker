# Guitar Tab Maker
easy way to write and store guitar tabs.
just **click** the fret you want to play.
**each step denotes __one beat__**, this is so that multiple notes can also be regestered at once.
curretly only gives a textual log of the notes played and has no memory.

## upcomming features
- proper tab notations for the notes instead of just text logs.
- visually better fretboard.
- memory to save previous notes and export notes.
